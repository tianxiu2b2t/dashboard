    帮助信息 - 华为应用市场看板使用说明
    📊 数据来源说明

    本看板数据收集自华为应用市场（AppGallery）公开API，数据仅供参考，不保证来源的准确性、完整性和真实性。

    🚀 主要功能
    • 数据概览 - 显示已爬取总数、应用总数、元服务总数、开发者总数
    • 总下载榜 - 展示应用下载量排行榜
    • 评分分布 - 显示应用评分的分布情况
    • 非华为应用下载榜 - 展示非华为应用和内置应用的下载排行
    • 应用列表 - 详细的应用信息表格，支持搜索、排序和快速筛选
    📝 使用说明
    • 刷新数据 - 点击"刷新数据"按钮获取最新数据
    • 搜索应用 - 在应用列表上方的搜索框中输入关键词，支持应用名称、包名、开发者等多种搜索方式
    • 快速筛选 - 点击应用列表中的开发者名称可快速搜索该开发者的所有应用（精确匹配）
    • 排序功能 - 点击表格标题（开发者、分类、评分、下载量、大小、上次更新）进行排序
    • 查看详情 - 点击应用行查看详细的应用信息
    • 过滤选项 - 使用"排除华为"和"排除元服务"复选框快速过滤应用
    • 分页浏览 - 使用底部分页控件浏览更多应用
    ⚠️ 注意事项
    • 部分功能仍在开发中，可能会出现bug或数据不完整的情况
    • 数据更新可能存在延迟，请以官方数据为准
    • 应用详情页面仍在施工中，部分数据可能显示不正确
    • 使用ESC键可以快速关闭弹窗
    🔧 技术支持

    如遇到问题或需要技术支持，请联系相关开发人员。

    搜索功能帮助
    🔍 搜索类别说明
    • 应用名称 (name): 搜索应用的显示名称，如 "微信"。
    • 包名 (pkg_name): 搜索应用的唯一包标识符，如 "com.tencent.mm"。
    • 应用 ID (app_id): 搜索华为应用市场的应用唯一 ID。
    • 开发者 ID (dev_id): 搜索开发者的唯一标识符。
    • 开发者名称 (developer_name): 搜索中文开发者名称。
    • 开发者英文名称 (dev_en_name): 搜索英文开发者名称。
    • 应用描述 (description): 搜索应用的描述信息。
    • 供应商 (supplier): 搜索应用的供应商信息。
    • 类型名 (kind_name): 搜索应用的类型名称。
    • 应用分类 (kind_type_name): 搜索应用的分类类型名称。
    • 标签名 (tag_name): 搜索应用的标签名称。
    ⚡ 精确匹配开关

    开启 (勾选): 进行精确匹配，只返回完全匹配关键词的结果。
    关闭 (未勾选): 进行模糊搜索，支持部分匹配，提高搜索灵活性。
    默认关闭（模糊搜索）。


    高级说明
    搜索使用 PostgreSql 的 ILIKE, 模糊搜索就是在搜索内容两边各加一个 %,
    因此你可以使用 % 来做通配符 _ 来做单个通配符
    或者使用其他 pg ILIKE 语法允许的特殊内容 (不用想着注入了, 我用的 .bind)

    ⚡ 快速筛选功能

    在应用列表中，开发者名称以浅蓝色按钮形式显示。
    点击开发者名称可快速搜索该开发者的所有应用，自动设置为精确匹配模式。
    这是一种快捷的筛选方式，无需手动输入搜索内容。

    📝 使用提示
    • 点击"搜索"按钮或按 Enter 键执行搜索。
    • 点击"清除搜索"按钮可一键清除所有搜索条件，恢复默认显示。
    • 搜索仅在输入关键词和选择类别后生效。
    • 支持排序和分页，与搜索结果结合使用。
    • 使用"排除华为"和"排除元服务"复选框可快速过滤特定类型的应用。
